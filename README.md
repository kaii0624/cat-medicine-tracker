# ムクカプセル

猫のお薬を朝・夜の1日2回記録し、家族の端末と共有する1画面のWebアプリです。

## できること

- 今日までの直近4日を朝・夜に分けて表示
- スクロール式の時刻選択で記録・更新・削除
- 前回からの経過時間を自動表示
- 家族用リンクから複数端末で同じ記録を閲覧
- Cloudflare D1へ記録を保存
- ログイン画面なし（あなた用リンクだけ記録可能）

## ローカル起動

依存パッケージをインストールし、Cloudflareのローカル環境を起動します。

```sh
npm install
npm run db:local
npm run dev
```

ローカル用の `VIEW_TOKEN` と `ADMIN_TOKEN` は、Git管理外の `.dev.vars` に設定します。

## 公開構成

- Cloudflare Worker: 画面配信とAPI
- Cloudflare D1: 投薬記録の共有保存
- あなた用リンク: 閲覧・記録・更新・削除
- 家族用リンク: 閲覧のみ

権限キーはURLのフラグメントから各端末に保存され、サーバーのアクセスログやGitHubリポジトリには含めません。
