# ムクカプセル

猫のお薬を朝・夜の1日2回記録する、1画面のシンプルなWebアプリです。

## できること

- 今日までの直近4日を朝・夜に分けて表示
- スクロール式の時刻選択で記録・更新・削除
- 前回からの経過時間を自動表示
- 端末の共有メニューからLINEなどへ最新記録を共有
- 記録をブラウザ内に保存

## 起動

このフォルダでローカルサーバーを起動し、ブラウザで開きます。

```sh
python3 -m http.server 4173
```

その後、`http://localhost:4173` を開いてください。

## 保存・共有について

記録はブラウザのローカルストレージに保存されます。複数端末へ自動同期はしません。「家族に共有」ボタンは最新記録を共有用メッセージとして送ります。
